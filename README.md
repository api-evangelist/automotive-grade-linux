# Automotive Grade Linux (automotive-grade-linux)
Automotive Grade Linux (AGL) is a collaborative open source project under the Linux Foundation developing a shared software platform for the automotive industry. It provides a Linux-based operating system stack for in-vehicle infotainment, telematics, instrument clusters, and connected car applications.
